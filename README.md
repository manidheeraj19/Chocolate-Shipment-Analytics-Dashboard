# Chocolate Shipment Analytics Dashboard 🍫📊
This project focuses on building an interactive Power BI dashboard to analyze chocolate shipment data and gain insights into sales performance, regional distribution, and product demand. The dashboard helps visualize key business metrics and enables data-driven decision making through dynamic charts and filters.


## Overview
The **Chocolate Shipment Analytics Dashboard** is a data visualization project built using **Power BI** to analyze chocolate shipment data across different regions, products, and sales channels. The dashboard provides insights into shipment trends, product demand, and overall sales performance to support data-driven decision making.

## Project Objectives
- Analyze chocolate shipment data to identify sales trends.
- Track product performance across different regions.
- Visualize shipment distribution and revenue metrics.
- Build interactive dashboards for easy data exploration.

## Features
- Interactive **Power BI dashboard** with dynamic filters and slicers.
- Visualization of **shipment trends, product categories, and regional performance**.
- Key performance indicators (KPIs) such as total shipments, revenue, and top-selling products.
- Data transformation and cleaning using **Power Query**.
- Calculated measures and metrics using **DAX**.

## Tools & Technologies
- **Power BI**
- **Power Query**
- **DAX**
- **Microsoft Excel / CSV dataset**

## Dashboard Insights
The dashboard provides insights into:
- Total shipments and revenue distribution
- Region-wise chocolate sales performance
- Top-performing chocolate products
- Shipment trends over time
